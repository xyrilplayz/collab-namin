# Project Description
COZYCUP Coffee House is a community-focused caf√© dedicated to serving high-quality, freshly brewed coffee in a cozy and inviting atmosphere. Our mission is to create a space where coffee enthusiasts can relax, socialize, and enjoy expertly crafted beverages.
